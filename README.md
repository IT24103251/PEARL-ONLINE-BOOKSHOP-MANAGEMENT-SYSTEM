# Online Bookshop Management System

This project is an **Online Bookshop Management System** developed as part of the Information Systems Project 2026 at the **Sri Lanka Institute of Information Technology (SLIIT)**. The system is designed to transition traditional, manual bookshop operations into a centralized, automated platform to improve efficiency and customer experience.

## 📖 Project Overview

Traditional bookshops often struggle with manual record-keeping, leading to errors in inventory, misplaced orders, and slow supplier communication. This system addresses these challenges by providing:

* 
**Customer Management:** Automated registration, profile updates, and loyalty classification.


* 
**Inventory Tracking:** Real-time monitoring of stock levels, low-stock alerts, and automated catalog updates.


* 
**Order & Payment Processing:** Secure digital transactions and efficient order tracking from placement to delivery.


* 
**Supplier Coordination:** Management of purchase orders and supplier performance tracking.



---

## 🚀 Technical Stack

* **Frontend:** React + Vite (Fast Refresh enabled via @vitejs/plugin-react)
* **Project Structure:** Built with HMR (Hot Module Replacement) and ESLint for code quality.

## 📋 Project Status (Sprint 1 Preview)

The goal of the current phase (Sprint 1) is to deliver a functional core version of the system. Key features currently being implemented include:

* Secure payment gateway integration.


* Basic book recording and categorization in inventory.


* Customer order placement and admin order confirmation.


* Delivery address management and status tracking.

## 🛠️ Getting Started

To get the development environment running locally:

1. **Clone the repository**
2. **Install dependencies:**
```bash
npm install

```


3. **Run the development server:**
```bash
npm run dev

```
